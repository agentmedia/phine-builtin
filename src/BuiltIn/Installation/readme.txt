= About the BuiltIn Bundle =

The BuiltIn bundle provides basic modules for integration in Websites of the
Phine CMS. This bundle is part of the Phine CMS, maintained and develeoped by 
its core team.

== Version History ==

=== 1.0.0 ===

First version including the following elements.

* content block
* html articles
* html code
* html wraps
* navigation with individual items

=== 1.0.1 ===

The navigation now uses the page url that is provided by the core bundle since version 1.0.1.
Frontend member support; new registration elements brought to backend:

* register simple
* register confirm
* login
* logout
* lost password

=== 1.0.2 ===

Added the useful repeater for easily iterating over child content elements
